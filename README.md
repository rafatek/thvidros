# THvidros - Website

Site profissional para vidraçaria THvidros desenvolvido com HTML, CSS e JavaScript puro.

## Características

- Design responsivo (mobile-first)
- Navegação intuitiva
- Seções: Hero, Sobre, Serviços, Diferenciais, Galeria e Contato
- Botão WhatsApp flutuante
- Formulário de contato integrado com WhatsApp
- Animações suaves
- Menu mobile funcional
- Paleta de cores profissional (azul e cinza)

## Estrutura de Arquivos

\`\`\`
thvidros-website/
├── index.html      # Arquivo principal HTML
├── styles.css      # Estilos CSS
├── script.js       # Funcionalidades JavaScript
└── README.md       # Este arquivo
\`\`\`

## Como Usar

1. Baixe os arquivos do projeto
2. Abra o arquivo `index.html` em um navegador
3. Para fazer upload em um servidor:
   - Faça upload de todos os 3 arquivos (index.html, styles.css, script.js) para o servidor
   - Certifique-se de que estão no mesmo diretório
